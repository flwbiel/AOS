Fonte do Problema

A nossa parceira Uol Diveo gera grandes quantidades de dados distintos em tempo real que hoje são armazenados em diversas bases de dados diferentes. Isso dificulta a realização de análises sobre essas informações.

Solução

A solução que nós desenvolvemos em conjunto a Uol foi agrupar todos esses dados distintos em uma base única (um data lake).

Processamento em Lotes

Para isso, desenvolvemos um sistema de processamento em lotes. Esse sistema consome os dados de todas as bases separadas, persitindo os dados consumidos em nosso data lake

Integridade dos Dados

O sistema realiza a leitura sobre esses dados e garante sua integridade, atualizando qualquer registro que for alterado em uma das diferentes bases.

Dashboard

Para exemplificação do potêncial do data lake, disponibilizamos um dashboard com análises superfíciais sobre esses dados.
